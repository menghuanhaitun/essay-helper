# 李欣阳的作文助手

## 介绍
这是李欣阳专用的作文辅助工具，包含6大功能模块。

## 文件说明
- `index.html` - 主程序（包含所有功能逻辑）
- `vocab-data.js` - 好词好句库数据文件（独立维护）

## 功能模块
1. 📚 好词好句库 - 6大场景分类，随机展示
2. 🧭 作文引导器 - 4种作文类型的写作框架
3. ⬆️ 句子升级器 - 将普通句子升级为优美句子
4. 🔍 范文拆解 - 优秀段落技法分析
5. ✏️ 写作练习 - 智能批改写作练习
6. 🏆 荣誉墙 - 积分系统和成就系统

## 部署方式
- 本地使用：直接双击 `index.html` 打开
- 在线访问：部署到 Cloudflare Pages 后获得访问链接

## 技术栈
纯 HTML + CSS + JavaScript，无任何依赖，无需后端服务器。

## 数据存储
- 积分和自定义好词好句保存在浏览器的 localStorage 中
- key: `lxyEssayHelper`
